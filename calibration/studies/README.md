# Study PDFs

Place your original cost segregation study PDFs here. They are gitignored and will not be committed.

## Naming Convention

Use descriptive names:
- `stella_8222_calmont_purchase_2022.pdf`
- `cypress_edge_1565_ih35_reno_2023.pdf`
- `queen_dr_3404_cedar_rapids_2024.pdf`

## Subfolders

- `purchase/` — Full-property acquisition studies
- `renovation/` — Renovation/improvement-only studies
- `supplemental/` — Excel files, asset lists, supporting docs
